## 实现的一些效果

* UI相关
* ListView相关
* GridView相关
* 弹框封装(中间、底部、toast、自定义弹框)
* 仿微信右上角弹框
* 时间日期单列多列底部选择器
* 拍照，本地图片多选，九宫格展示，全屏查看图片
* JhForm(表单录入)
* dio封装，分页上下拉加载数据
* baseNav、baseAppBar、BaseTabBar、BaseWebView、封装
* 导航条搜索框
* chart相关（多图表，渐变折线图）
* JhTopTabBar(导航条分页切换)
* 小红点
* 二维码扫描与生成
* 轮播
* Animation（标签云）
* 上下拉刷新、分组吸顶ListView
* 倒计时按钮
* 登录注册
* 版本新特性
* 检查更新
* 路由管理
* 封装一些组件和工具类(在jh_common文件夹)
